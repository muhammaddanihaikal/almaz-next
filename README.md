# Almaz Next

Proyek manajemen stok dan distribusi menggunakan Next.js.

## Fitur Utama
- **Dashboard**: Visualisasi data transaksi.
- **Manajemen Distribusi**: Pencatatan barang keluar ke toko.
- **Manajemen Retur**: Pengelolaan barang kembali dari toko.
- **Ekspor Excel**: Laporan dalam format Excel dengan styling khusus (xlsx-js-style).
- **Desain Responsif**: Antarmuka yang dioptimalkan untuk perangkat seluler.

## Teknologi
- **Framework**: Next.js (App Router)
- **Database**: PostgreSQL (Neon) dengan Prisma ORM
- **Styling**: Tailwind CSS
- **Autentikasi**: NextAuth.js
